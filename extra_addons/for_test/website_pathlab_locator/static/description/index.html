Pathology Lab Center Locator

Pathology Lab Center Locator is a module by which customers can locate the collection centers of a Pathology on google map. Therefore, customers can find the nearby collection centers of a pathology and get the directions for the center on google map. Also, customer can check lab timings, images and facilities provided in a particular lab center.

Highlighted Features

  • Highlights location of all the collection centers of the pathology in google map.
  • Locate all centers of pathology on Google Map using marker.
  • Admin can set configuration for the default map.
  • Customers can also search for the collection center in the provided search box with the map view.
  • Customers can get the contact number, email, address of the collection centers available.
  • Customers can get the directions to a collection center.
  • Customers can check lab timings, images and facilities provided in a particular lab center.
